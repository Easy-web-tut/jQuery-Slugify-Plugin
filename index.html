As you type the title of a page...

the slug is created automatically.

Your URL is http://geocities.com/dhtml/.

Could it be any simpler than this?

$('.slug').slugify('#title');

Don't like default slug format? Pass in your own slugFunc.

$('#caps_slug').slugify('#title', {
	slugFunc: (function(val, originalFunc) { return (originalFunc(val).toUpperCase()); } )
});
